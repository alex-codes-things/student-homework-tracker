# Day 3 - Form Submission & HTML Appending

Today I started quite early and spent around 1 hour learning and practising JavaScript.

The main focus for Day 3 was understanding how form submission works and how JavaScript can be used to take information from a form and display it on the page.

## What I Worked On

I started by learning how to capture data from a form when the submit button is clicked. I also learned how to stop the page from refreshing when the form submits, which is important because normally a form submission reloads the page by default.

After that, I practised logging the form data to the console so I could check if the values were being collected correctly.

Once I understood that part better, I moved on to creating a new task card using JavaScript. The form now takes the submitted data and adds a new HTML task card into the task container.

## What I Learned

- How to listen for a form submit event
- How to prevent the page from refreshing after submitting a form
- How to get values from form inputs
- How to test form data using the console
- How to create and add a new task card to the page with JavaScript
- How JavaScript can connect form input with visible HTML elements

## Reflection

This might not seem like a huge amount of progress, but because I am still very new to JavaScript, this took longer than expected.

I am starting to understand how JavaScript can make a website feel more interactive. Today was an important step because the project is no longer just static HTML and CSS — it can now respond to user input.
