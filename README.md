# Student Homework Tracker

A practical student dashboard for tracking homework tasks, due dates, priorities, completion status, and overdue work.

This project is being built as a beginner-friendly web development project using HTML, CSS, and JavaScript. The goal is not only to create a useful homework tracker, but also to practice real-world project planning, version control, layout design, and step-by-step development.

---

## Project Goal

The goal of this project is to create a simple and useful dashboard that helps students keep track of their homework.

The app will allow a student to add homework tasks, view important task details, mark work as completed, and clearly see when a task is overdue.

This project is also being used to practise building a project from the planning stage all the way to a finished version.

---

## Target User

This project is designed for students who want a simple way to manage homework without needing a complicated app.

The ideal user is someone who wants to quickly see:

- What homework they have
- Which subject it belongs to
- When it is due
- How important it is
- Whether it has been completed
- Whether anything is overdue

---

## Core Features

The first version of the project will include:

- Add a homework task
- Add a subject
- Add a due date
- Add a priority level
- Mark a task as completed or not completed
- Display homework tasks on the dashboard
